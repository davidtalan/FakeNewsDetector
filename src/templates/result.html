Interpreting the results

Since the training data used for this project mainly consisted of politically-related articles, it may be the case that reliable articles talking about political topics gets flagged as unreliable. The similarity score is shown to either support or dispute the result of the machine learning model. The more articles found on Google talking about the same thing as the target article, the more reliable it would be. The opposite could be said if there are very few results that talk about the same topic.
